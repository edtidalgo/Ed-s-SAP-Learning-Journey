# Ed's SAP Learning Journey

Welcome to my SAP learning portfolio! 

This repository documents my progress in mastering **SAP ERP functional skills** with a focus on **MM, SD, and FI/CO modules**.  
It also serves as a knowledge base and showcase for my career transition back into IT.

---

## Learning Goals
- Build strong foundational knowledge of ERP concepts and SAP navigation.
- Gain functional expertise in key SAP modules:
  - **MM (Materials Management)** → Procurement, inventory, vendor management
  - **SD (Sales & Distribution)** → Order processing, billing, shipping
  - **FI/CO (Finance & Controlling)** → General ledger, accounts payable/receivable
- Document workflows, processes, and integration points.
- Share insights publicly to help others and demonstrate growth.

---

## Repository Structure
- `/notes` → Markdown notes for each module (MM.md, SD.md, FI.md, CO.md)
- `/process-flows` → Diagrams of ERP workflows (Procure-to-Pay, Order-to-Cash, Record-to-Report)
- `/resources` → Links to courses, articles, and references
- `/scripts` → Any automation or practice scripts (Python/PowerShell for ERP-related tasks)
